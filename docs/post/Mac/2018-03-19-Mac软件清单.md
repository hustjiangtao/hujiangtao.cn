## Mac必备

这里的是必备的软件，有些是命令行工具，有些是软件，无论是对于刚接触Mac的小白，还是经验丰富的老手，都是居家旅行撸码必备的神器。

- iTerm2

```
一个代替系统Terminal的终端工具，Amazing，从此可以抛弃自带终端愉快的敲命令啦
官网：http://www.iterm2.com/
App Store：无
推荐brew安装：brew install iterm2
```

- oh~my~zsh

```
终端美化，终端增强，把你的命令行帅到家、美到爆，多种现成的主题供你选择，更可以每次自动随机一款主题
官网：http://ohmyz.sh/
App Store：无
```

- HomeBrew

```
macOS 缺失的软件包管理器，方便安装各种命令行工具和软件，通常和brewcask搭配食用更佳，brewcask是类似于homebrew通过命令行安装桌面软件的工具
官网：https://brew.sh/
App Store：无
```

- Alfred

```
效率神器，尤其是新款的Alfred3，添加了更多功能，尤其是书签的搜索也是很方便的啊
官网：https://www.alfredapp.com/
App Store：有
```

- Google Chrome

```
Safari也可以，但是重度使用还是没有Chrome来的方便，尤其对于开发者来说，调试不要太方便，还有一个小秘密，很多开发者基本都是首先默认基于Chrome来开发的...，然后才会考虑其他浏览器的适配
推荐brew安装：brew cask install google-chrome
```

- AppCleaner

```
好用的卸载工具，卸载较为彻底
官网：http://freemacsoft.net/appcleaner/
App Store：无
推荐brew安装：brew cask install appcleaner
```

- Keka

```
好用的压缩工具，官网免费，App Store收费
官网：http://www.kekaosx.com/en/
App Store：有
```

```
官网：https://www.sublimetext.com/
App Store：无
```

附带配置教程：[sublime 配置教程](https://www.hujiangtao.cn/post/Mac/2018-05-07-Sublime%20Text%E9%85%8D%E7%BD%AE%E5%92%8C%E4%BD%BF%E7%94%A8.html#renference)

- PyCharm

```
作为Python开发最好用的IDE，没有之一，当然小项目或者脚本，其他的一样胜任
推荐brew安装：brew cask install pycharm
```

- Fork

```
好用的轻巧的Git客户端可视化工具，用了都说好
官网：https://git-fork.com/
App Store：无
推荐brew安装：brew cask install fork
```

- Sequel Pro

```
轻量级的MySQL客户端
官网：http://www.sequelpro.com/
App Store：无
推荐brew安装：brew cask install sequel-pro
```

- DataGrip

```
数据库管理工具，看数据比较方便，数据导入导出功能不太好用，个人按需选择吧，本人目前已不用它了，因为启动太慢了，现在一直在用上面的 Sequel Pro，轻量方便，也对sql语句更友好，但是仅限于MySQL
官网：https://www.jetbrains.com/datagrip/
App Store：无
推荐brew安装：brew cask install datagrip
```

- MySQLWorkbench

```
MySQL官方客户端，导入导出数据和定义数据库结构好用，但查看数据不好用，结合DataGrip和Seqel Pro使用更佳
官网：https://dev.mysql.com/downloads/workbench/
App Store：无
```

- Medis

```
Redis GUI 管理软件，觉得还行，一直在用
官网：http://getmedis.com/
```
